About Me

Full Stack Developer en formación con enfoque en desarrollo web moderno. Apasionado por el frontend, la lógica backend y el diseño visual. Siempre aprendiendo, siempre creando.

Técnico en Desarrollo de Software con formación práctica en el SENA y actualmente en entrenamiento intensivo en CampusLands. Cuento con experiencia en desarrollo web utilizando HTML, CSS, JavaScript y consumo de APIs REST para construir interfaces dinámicas. Manejo proyectos con estructura modular (HTML, CSS, JS separados), integración de datos externos y control de versiones con Git y GitHub. Tengo conocimientos en backend con Python, incluyendo estructuras básicas de servidores, manejo de rutas, y lógica de negocio. También he trabajado con WordPress a nivel intermedio para la creación y personalización de sitios web. Además, tengo bases en Tailwind CSS, maquetación responsive, desarrollo frontend moderno y automatización de tareas. Me destaco por mi aprendizaje autodidacta, capacidad de análisis, resolución de problemas y compromiso con la mejora continua como desarrollador web full stack junior.
